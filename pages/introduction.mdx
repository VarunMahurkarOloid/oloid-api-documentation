# Welcome to the Oloid API

Thanks for choosing Oloid! This guide will help you quickly get started with integrating the Oloid API into your application.

All requests to the Oloid API require authentication using an API key.

### Get Your API Key

1. Log in to your Oloid Dashboard.
2. Navigate to **API Settings**.
3. Generate or copy your API key.

Once you have your API key, include it in the `Authorization` header of your requests:

```bash
Authorization: Bearer YOUR_API_KEY
```
